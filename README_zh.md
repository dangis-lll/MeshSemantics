# MeshSemantics

[English README](./README.md)

MeshSemantics 是一个用于三角网格语义标注的桌面软件。

它基于 `Python + PyQt6 + vedo + VTK` 开发，适合用于批量打开 `STL` / `VTP` 网格模型，对三角面片进行交互式标注，并将结果导出为后续训练、分析或处理所需的数据格式。

## 软件功能

- 打开单个网格文件，或扫描整个项目文件夹
- 在三维视图中交互式查看三角网格
- 通过以下方式给三角面片打标签：
  - 右键单面片选择
  - 基于样条轮廓的曲面框选
- 在真正应用标签前先预览待标注区域
- 支持撤销 / 重做
- 支持快速保存当前任务为 `VTP`
- 支持导出：
  - 含标签数据的单个 `VTP`
  - 按标签拆分的多个 `STL`
- 支持项目任务完成状态管理

## 支持的文件格式

- 输入：
  - `*.stl`
  - `*.vtp`
- 输出：
  - `*.vtp`
  - `*.json`
  - 按标签拆分的 `*.stl`

## 界面组成

- 中央区域：三维网格视图
- 左侧面板：项目文件列表
- 右侧面板：标签面板、颜色映射、快速保存、任务完成状态
- 顶部工具栏：打开、保存、撤销、重做
- 底部状态栏：显示当前模式和交互反馈

## 标注方式

### 1. 单面片选择

在模型上右键点击，可以切换单个三角面片的选择状态。

- 如果该面片当前未选中，则加入选择
- 如果该面片当前已选中，则取消选择
- 按 `E`，把当前选中的面片赋予当前标签

### 2. 样条曲面框选

按 `S` 进入样条模式。

在样条模式下：

- 左键单击：在模型表面添加控制点
- 左键单击第一个控制点附近：闭合轮廓
- 左键单击预览曲线：在线上插入新的控制点
- 左键拖动：旋转模型
- 中键拖动：保留 VTK 默认相机交互
- `Delete` 或 `Backspace`：删除当前高亮的控制点
- `Enter`：根据闭合轮廓计算预览区域
- `E`：应用当前预览区域
- `C`：清除当前样条并退出预览 / 编辑流程

当前样条框选采用的是“曲面轮廓 + 曲面裁剪”的逻辑：

1. 控制点从模型表面拾取
2. 根据控制点生成样条曲线
3. 将样条点重新吸附到模型表面
4. 使用闭合曲面轮廓对网格进行裁剪
5. 取裁剪结果中的最大连通区域作为预览选择

## 快捷键

| 快捷键 | 功能 |
| --- | --- |
| `S` | 进入样条模式 |
| `Enter` | 确认样条轮廓并生成预览 |
| `E` | 将当前预览区域应用到当前标签 |
| `C` | 清除当前预览 / 退出样条流程 |
| `M` | 切换任务完成状态 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Y` | 重做 |
| `Delete` / `Backspace` | 删除高亮的样条控制点 |

## 推荐使用流程

1. 打开一个文件或项目文件夹。
2. 在右侧面板选择目标标签。
3. 用右键单选或样条曲面框选选择需要标注的区域。
4. 按 `E` 应用标签。
5. 重复上述操作，完成其他区域标注。
6. 点击 `Quick Save` 或另存为 `VTP`。
7. 需要时导出按标签拆分的 `STL`。
8. 将任务标记为完成。

## 运行方式

在仓库根目录运行：

```bash
python main.py
```

## 建议环境

推荐 Python 版本：

- Python 3.10 及以上

项目当前依赖的核心第三方库：

- `PyQt6`
- `vedo`
- `vtk`
- `numpy`

如果你要新建环境，典型安装方式如下：

```bash
pip install -r requirements.txt
```

或者：

```bash
pip install PyQt6 vedo vtk numpy
```
